Browser Edition vs. Traditional Workflow

Browser Edition (New!)

  • ✅ No software installation needed
  • ✅ Generate STL files directly
  • ✅ Instant preview of your designs
  • ✅ Works on any modern browser
  • ❗ More resource-intensive for many names

Traditional Method

  • ❗ Requires Python and OpenSCAD installation
  • ❗ Multiple steps to generate files
  • ✅ More robust for large batches
  • ✅ More customization options
  • ✅ Lower computer resource usage
Switch to Traditional Method

How It Works

  1. Enter your names (one per line)
  2. Click "Preview" to see how the first name will look
  3. Click "Validate Names" to check all names for errors
  4. Click "Generate STL Files" to create downloadable 3D models
  5. Send the STL files to your 3D printer
Browser Compatibility Error
Your browser might not support the 3D libraries needed for this feature. Please try:
  • Using a modern browser like Chrome, Firefox, or Edge
  • Enabling WebGL in your browser settings
  • Disabling ad blockers or script blockers for this site
  • Using the traditional method instead
