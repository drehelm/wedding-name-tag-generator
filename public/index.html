Name Guidelines:

  • Names are automatically converted to uppercase
  • Maximum 14 characters per name (shorter names display better)
  • Only letters, numbers, spaces and hyphens allowed
  • One name per line

How It Works

  1. Enter your names (one per line)
  2. Click "Validate Names" to check for errors
  3. Click "Generate Name Tags" to create and download the ZIP file
  4. Follow the included instructions to create your 3D printable name tags

Requirements

To use the generated files, you'll need:

  • OpenSCAD - For generating 3D models
  • Python 3 - For running the script
  • STIX Two Text Font - Required for the text style

Note: If you prefer not to install these programs, try our Browser-Only Edition instead!
